Odds, Odds Ratios, Confidence Intervals, and Statistical Estimation

A visual revision sheet for the core logic: what odds are, how odds ratios are built and interpreted, what confidence intervals really mean, and why estimation is about both accuracy and precision.

Odds event / non-event Compare how often something happens to how often it does not happen.
Odds Ratio (a/b) / (c/d) Compare the odds in one group to the odds in another.
Confidence Interval estimate +/- uncertainty Shows a plausible range for the true population value.

1. Probability vs Odds

Students often mix these up. Probability is about the share of times an event happens out of all possible outcomes. Odds compare the number of events to the number of non-events.

Probability

How likely is the event out of all cases?

Probability = event / total
30 events70 non-events
30%
70%

Example: if 30 out of 100 people have the disease, probability = 30/100 = 0.30.

Odds

How likely is the event compared with the non-event?

Odds = event / non-event
30 events70 non-events
30
70

Using the same example: odds = 30/70 = 0.43, often read as roughly 3 to 7.

Shortcut: probability uses the whole denominator. Odds use only the opposite outcome in the denominator.

2. Converting Between Probability and Odds

The same information can be written either way. The important thing is to know which form the question is using.

From probability to odds

Odds = p / (1 - p)

If probability is 0.20:

Odds = 0.20 / 0.80 = 0.25

So the odds are 1:4.

From odds to probability

Probability = odds / (1 + odds)

If odds are 3:

Probability = 3 / 4 = 0.75

So the event probability is 75%.

Odds Ratio = (a / b) / (c / d) = ad / bc

Worked example

Suppose:

  • a = 40
  • b = 60
  • c = 20
  • d = 80
OR = (40 x 80) / (60 x 20) = 3200 / 1200 = 2.67

The exposed group has about 2.67 times the odds of the outcome.

Interpretation guide

OR < 1 protective
OR = 1 no association
OR > 1 harmful/risk
  • OR = 1: no difference in odds between groups.
  • OR > 1: higher odds in the exposed group.
  • OR < 1: lower odds in the exposed group.
Important: odds ratio is not the same as risk ratio. They are close only when the outcome is rare.

4. Confidence Intervals

A confidence interval gives a range of plausible values for the true parameter. It combines a point estimate with its uncertainty.

Anatomy of a confidence interval

Confidence Interval = Point Estimate +/- Margin of Error

The point estimate is your best single-number summary from the sample. The interval reflects sampling variability.

  • Narrow interval = more precision
  • Wide interval = less precision
  • Larger sample sizes usually narrow the interval

Visual intuition

Study A: OR = 0.72 [0.58, 0.89]
Study B: OR = 0.95 [0.74, 1.18]
ProtectiveNull = 1Risk
Correct interpretation of a 95% CI: if we repeated the sampling process many times and built a 95% CI each time, about 95% of those intervals would contain the true population parameter.

5. The Null Value Rule

For interpretation, you always compare the confidence interval with the null value. The null value depends on the measure.

Ratio Measures

Null value = 1

Odds ratio, risk ratio, hazard ratio.

If the CI includes 1, the data are compatible with no association.

Difference Measures

Null value = 0

Mean difference, risk difference, regression coefficient.

If the CI includes 0, the data are compatible with no difference.

Exam Logic

Significance check

For ORs, a CI entirely above or below 1 suggests statistical significance at the 5% level.

6. Principles of Statistical Estimation

Estimation is the process of using a sample to learn about a population parameter. A good estimator should not just give a number. It should give a defensible number.

Point estimate A single best estimate, such as a sample mean or odds ratio.
Bias Does the estimator systematically miss the true value in one direction?
Precision How much do estimates vary from sample to sample?
Consistency Does the estimator get closer to the true value as sample size grows?
Efficiency Among unbiased estimators, does it have smaller variance?

Bias vs precision

These are different ideas. An estimator can be unbiased but imprecise, or precise but biased.

  • Biased: average estimate is off target.
  • Precise: repeated estimates are tightly clustered.
  • Best case: low bias and high precision.

Standard error

The standard error measures how much an estimate tends to vary from sample to sample. It is a key ingredient in confidence intervals.

Estimate +/- critical value x standard error

Smaller standard error means a narrower interval and more precise estimation.

7. Worked Interpretation Examples

These are the kinds of statements examiners usually want.

Example A

OR = 1.80 [1.20, 2.70]

The exposed group has higher odds of the outcome. Because the CI does not include 1, the association is statistically significant.

Example B

OR = 0.65 [0.42, 0.98]

The exposure appears protective. The CI stays below 1, so the association is statistically significant.

Example C

OR = 1.30 [0.84, 2.01]

Although the point estimate suggests higher odds, the CI includes 1, so the result is not statistically significant.

8. Common Mistakes

These are the errors that keep showing up in revision and exams.

Confusing odds with probability

Odds use non-events in the denominator. Probability uses total observations.

Reading OR as absolute risk

An OR of 2 does not mean the probability doubled.

Ignoring the CI

The point estimate alone is not enough. Always check the interval.

Using the wrong null value

Ratios use 1. Differences use 0.

Calling a wide CI strong evidence

Wide intervals usually mean low precision, often from a small sample.

Misstating the 95% CI

It is not correct to say there is a 95% chance the fixed true value lies in this one computed interval.

9. Exam Answer Flow

A compact structure for answering interpretation questions.

1

Name the measure

State whether it is an odds ratio, risk ratio, mean difference, or something else.

2

Interpret the direction

Above 1 means higher odds, below 1 means lower odds for ORs.

3

Check the confidence interval

Ask whether it crosses the null value.

4

Comment on precision

Narrow intervals suggest greater precision than wide intervals.

10. Questions and Answers for Recap

Open each question and test whether your explanation matches the answer.

Q1. What is the difference between probability and odds?
Probability is the number of events divided by the total number of observations. Odds are the number of events divided by the number of non-events.
Q2. If probability is 0.25, what are the odds?
Odds = 0.25 / 0.75 = 0.33, or about 1 to 3.
Q3. If odds are 4, what is the probability?
Probability = 4 / (1 + 4) = 4/5 = 0.80.
Q4. What does an odds ratio of 1 mean?
It means the odds are the same in both groups. There is no association on the odds scale.
Q5. What does an odds ratio below 1 mean?
It means the exposure is associated with lower odds of the outcome. This is often described as a protective association.
Q6. Why is 1 the null value for odds ratios?
Because a ratio of 1 means both groups have equal odds. Any value above or below 1 implies a difference.
Q7. What does a 95% confidence interval try to show?
It gives a range of plausible values for the true population parameter, based on the sample and the sampling uncertainty.
Q8. If an OR is 1.4 with 95% CI [0.9, 2.1], is it statistically significant?
No. The interval includes 1, so the data are compatible with no association at the 5% level.
Q9. Why is a narrower confidence interval usually better?
It indicates more precision. The estimate is being pinned down more tightly.
Q10. What is the difference between bias and precision?
Bias refers to systematic error, where estimates tend to miss the true value in one direction. Precision refers to how much estimates vary from one sample to another.
Q11. What is a point estimate?
It is a single number from the sample used to estimate a population parameter, such as a sample mean, proportion, or odds ratio.
Q12. Why do larger samples often give better estimates?
Larger samples usually reduce sampling variability, giving smaller standard errors and narrower confidence intervals. That improves precision.
